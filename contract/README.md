## SlotChain
Decentralized Booking & Reservation System on Stellar

![Description of image](./public/image.png)

Contract address = CAA35ZY3ODUFMTZNHGD4BUTCEGYQJF5V5VHQ34DNA4OWPNRDRJTIWWP4


SlotChain is a decentralized application (dApp) that enables users to create, manage, and book time slots securely on-chain. Built on the Stellar network using Soroban smart contracts, it provides a trustless and transparent way to handle appointments and payments.

🚀 Features
⏱️ Create and manage service time slots

📅 Book appointments in a decentralized manner

💸 Secure on-chain payment settlement

🔐 Trustless interactions via smart contracts

⚡ Fast and low-cost transactions on Stellar

🧠 How It Works
Service providers create available time slots

Users browse and book slots

Payments are handled through smart contracts

Booking confirmations are recorded on-chain
